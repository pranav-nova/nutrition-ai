# 🥗 Nutrition AI Agent

An end-to-end **AI-powered nutrition label analyzer** that interprets food ingredients and nutrition facts, and evaluates the health impact using a **multi-agent architecture with LLM-as-Judge**.

---

## 🎯 Problem Statement

Food labels are often complex and difficult for everyday users to understand.  
Consumers struggle to interpret ingredient lists, additives, and nutritional values.

### 👤 Target Users
- General consumers
- Students
- Health-conscious individuals

### 💡 Solution

This project builds an **AI agent system** that:
- Explains ingredients in simple language
- Analyzes nutritional impact
- Evaluates output quality using an **LLM-based judge**
- Provides a final health conclusion

---

## 🧠 System Overview

This is a **multi-agent AI system**:

### 🔹 Agent 1 — Analysis Agent
- Takes food label input
- Explains ingredients
- Analyzes health impact
- Generates nutrition score

### 🔹 Agent 2 — Judge Agent (LLM-as-Judge)
- Evaluates output using a structured rubric
- Compares ingredient & health analysis
- Generates **final conclusion**

---

## ⚙️ Features

- 🧾 Single input for full food label (ingredients + nutrition)
- 🧪 Ingredient breakdown in simple language
- ⚡ Health impact analysis
- 📊 Nutrition score (0–100)
- 🧠 LLM-as-Judge evaluation system
- 🏁 Final conclusion generated by judge agent
- 🌐 Deployed web application

---

## 🏗️ Architecture

```

User Input (Streamlit UI)
↓
Analysis Agent (Groq LLM)
↓
Structured Output (Ingredients + Health + Score)
↓
Judge Agent (LLM-as-Judge)
↓
Final Evaluation + Conclusion
↓
Displayed on UI

```
